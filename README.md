Gold Proxy Checker

Инструмент для проверки прокси в формате login:password@ip:port. Поддерживает HTTP и SOCKS5.  
При запуске спрашивает тип прокси и проверяет их асинхронно с live-логированием.

Функции:
- Проверка до 100 HTTP-прокси или 50 SOCKS5-прокси одновременно(можете сами настроить количество проверяемых прокси)
- Мгновенная запись результата в файл после проверки каждого прокси
- Сортировка на три категории:
  - working.txt — рабочие прокси (корректно возвращают IP)
  - dead.txt — нерабочие (ошибки подключения, таймаут)
  - invalid.txt — подозрительные (например, HTTP/0.9 или некорректный ответ)
- Поддержка цветного вывода в консоли (если установлен colorama)

Требования:
- Python 3.8 или новее

Установка:
pip install -r requirements.txt

Использование:
1. Создайте файл input.txt и добавьте прокси по одному в строке:
   user1:pass1@1.2.3.4:8080
   user2:pass2@5.6.7.8:9999

2. Запустите скрипт:
   python main.py

3. Выберите тип прокси: HTTP (рекомендуется) или SOCKS5.
4. Дождитесь завершения. Результаты появятся в соответствующих файлах.
